# My Passwords Ver-0.018B

Mobile item-popup and folder-order polish.

## What changed
- Removed toast notifications for favourite/unfavourite actions.
- Removed the edit-cancel toast notification.
- On mobile, item popup Edit/Favourite/Delete actions now sit in one centred premium action row.
- On mobile, the Lock button moves to the top right beside the My Passwords title.
- Removed the homepage subtitle text.
- Vault and Settings buttons are centred on the same line on mobile.
- Clicking a folder no longer scrolls or jumps the page.
- Folder chips can be reordered by drag-and-drop on desktop.
- Folder chips can be reordered by long-press then drag/release on mobile.
- Folder order is remembered inside the encrypted vault metadata and syncs with the vault.

## Safety
- No Supabase SQL changes.
- No encryption changes.
- No sync engine changes.
- No dependency changes.
- Existing vault items remain backwards compatible.
