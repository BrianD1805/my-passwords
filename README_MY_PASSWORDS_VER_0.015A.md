# My Passwords Ver-0.015A

Vault-home UX patch.

## What changed
- Add item is now opened from a sticky floating plus button.
- Add/edit item form now opens in a popup rather than sitting on the homepage.
- Folder buttons now sit directly under the vault search.
- Favourites added as a folder.
- Vault items no longer show by default; user must search or choose a folder.
- Notes no longer show username/password fields.
- Checklists render as tickable checklist rows.
- Completed checklist items move to the bottom and show with strikethrough.

## Safety
- No Supabase SQL changes.
- No encryption changes.
- No sync engine changes.
- No dependency changes.
- No vault snapshot changes.
