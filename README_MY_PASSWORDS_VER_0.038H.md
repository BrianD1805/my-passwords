# My Passwords Ver-0.038H — Login password and key icon UX refinement

Focused visual and usability refinement for the vault login password field.

## Changes

- Reduced the rendered secure-device key icon from 29px to 26px, a further reduction of approximately 10%.
- Kept the lighter key stroke weight and transparent key button styling.
- Changed the master vault password field to show the entered password by default on the login screen.
- The eye control now hides the password when clicked and shows it again when clicked a second time.
- No secure-device unlock logic changes.
- No database changes.
