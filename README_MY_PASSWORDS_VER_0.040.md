# My Passwords Ver-0.040

## Settings UX redesign

- Replaced the Settings information dashboard and section-tab row with a clear grouped Settings directory.
- Added dedicated rows for My Account, Vault Safety, Emergency Access and Help & FAQs.
- Each substantial section opens as its own Settings page with a Back to Settings control.
- The vault Help icon now opens Help & FAQs directly rather than showing the Settings overview first.
- Removed the Vault Items, Safe, Verified and Ready information panels from Settings.
- Preserved the existing My Passwords colour scheme and all existing Settings functions.
- No database changes are required.
