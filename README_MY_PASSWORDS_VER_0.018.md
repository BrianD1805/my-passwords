# My Passwords Ver-0.018

User-created folders foundation.

## What changed
- Users can create their own folders from the folder row.
- Custom folders appear beside the built-in folders and Favourites.
- Add/edit item popup lets the user assign items to built-in or custom folders.
- Custom folder names are stored inside the encrypted vault data, not in a new database table.
- Folder creation uses the locked popup styling standard.

## Safety
- No Supabase SQL changes.
- No encryption changes.
- No sync engine changes.
- No dependency changes.
- Existing vault items are preserved.
